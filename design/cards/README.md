# Card artwork sources

Keep editable, high-resolution card artwork in this directory. Mirror the
shipped asset structure:

- `numbers/` — demigods and heroes
- `powers/` — gods and divine figures
- `icons/` — source vectors for effects
- `backs/` — card-back explorations

Export optimized files to `apps/client/public/assets/cards`. Do not place source
files such as PSD, Procreate, or uncompressed working images in the public app.
